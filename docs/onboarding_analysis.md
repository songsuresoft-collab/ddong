# SDV Intelligence Dashboard: Onboarding Tab 기능 상세 분석 리포트

본 문서는 **Onboarding** 탭 내에 구현된 세부 기능들의 UX 설계 의도, 기술적 구현 방식 및 사용자 경험 향상을 위한 특수 로직들을 심층적으로 분석합니다.

---

## 1. Onboarding Guideline (지능형 입사 가이드)
신입 사원이 조직에 안착하는 데 필요한 방대한 정보를 AI(NotebookLM)를 통해 대화형으로 탐색할 수 있는 서비스입니다.

### 핵심 기능 및 기술 상세
*   **AI 컨텍스트 대화 (Smart Chat)**: 
    - 입사 가이드 문서를 학습한 NotebookLM을 기반으로 실시간 질문에 답변합니다.
    - **가독성 최적화**: NotebookLM 특유의 인용구(ex: `[5-7]`)를 정규 표현식을 사용하여 실시간으로 제거함으로써 데이터의 정결함을 확보했습니다.
*   **지능형 질문 추천 (Smart Recommendations)**:
    - 시스템이 가이드 문서의 맥락을 분석하여 신입 사원이 궁금해할 만한 질문을 실시간으로 생성 및 제안합니다.
    - **UX 디테일**: 추천 질문 갱신 시 **스태거(Staggered) 애니메이션**을 적용하여 요소들이 순차적으로 나타나도록 시각적 역동성을 부여했습니다.
*   **시스템 분석 시각화**: 
    - 질문 분석 중에는 **스켈레톤(Skeleton) 로더**와 맥동(Pulsing) 효과가 적용된 안내 문구를 표시하여 사용자에게 '시스템이 지능적으로 분석 중'이라는 가시적인 피드백을 전달합니다.

---

## 2. My Progress (체크리스트 및 진척도 관리)
단순한 Todo 리스트를 넘어, 개인의 성취도 관리와 팀 내 적응 현황을 동시에 파악할 수 있는 대시보드입니다.

### 핵심 기능 및 기술 상세
*   **개인 온보딩 여정 (My Onboarding)**:
    - **진척도 시각화**: SVG 기반의 **프로그레스 링(Progress Ring)**을 사용하여 전체 완료율을 직관적으로 표시합니다.
    - **Milestone 구조화**: 과업을 '필수 프로그램', '보조 프로그램', '행정 절차' 등으로 카테고리화하여 로드맵 형태로 제공합니다.
*   **팀원 적응 현황 (Dept Status)**:
    - 부서원 전체의 온보딩 진척도를 한눈에 모니터링할 수 있는 리더용 뷰입니다.
    - **상세 분석 리포트**: 특정 팀원을 클릭 시 해당 인원의 과업 수행 내역을 정밀하게 분석하여 '임무 완수' 또는 '진행 중' 상태를 칩(Chip) 형태로 가시화합니다.

---

## 3. Self Growth (자기개발 및 교육 통합 관리)
사내 교육 리소스와 외부 컨퍼런스 정보를 통합하여 구성원의 지속적인 성장을 지원하는 서비스입니다.

### 핵심 기능 및 기술 상세
*   **사내교육 (Education Management)**:
    - **Google Sheets 연동**: 실제 운영 중인 구글 시트와 실시간 CRUD(생성, 조회, 삭제) 연동을 통해 데이터 관리의 연속성을 확보했습니다.
    - **기술적 해결**: 모달 UI가 부모 컨테이너에 의해 잘리는 문제를 해결하기 위해 **React Portals**를 도입하여 DOM 최상단에서 렌더링되도록 구현했습니다.
*   **컨퍼런스 (AI Conference Search)**:
    - **실시간 AI 검색**: Tavily와 NotebookLM을 연동하여 전 세계 최신 컨퍼런스 정보를 수집하고 구조화합니다.
    - **동기화 옵션 (Sync Mode Selection)**: 
        - **초기화 후 새 검색(Overwrite)** 및 **현재 내역에 추가(Append)** 모드를 드롭다운 메뉴를 통해 제공합니다. 
        - 이 과정에서 AI 소스 리셋 로직을 지능적으로 제어하여 데이터 중복을 방지합니다.
*   **프리미엄 로딩 UX**:
    - 모든 데이터 동기화 과정에 **글래스모피즘(Glassmorphism) 오버레이**와 브랜드 컬러 스피너를 적용하여 시스템의 프리미엄 감성을 유지합니다.

---

## 💡 기술적 우수성 요약
1.  **React Portal & Stacking Context**: 레이아웃 제약을 극복하는 전역 렌더링 시스템 구축.
2.  **Hybrid Data Source**: 직접 정의된 JSON 데이터와 구글 시트 API, AI 검색 엔진을 유연하게 통합.
3.  **Modern CSS Patterns**: CSS Modules 기반의 변수 활용 및 복합 키프레임 애니메이션을 통한 고도화된 UI 구현.
4.  **User-Centric Design**: 사용자의 실수 가능성을 차단하기 위한 드롭다운 옵션 및 경고 메시징 시스템 설계.
